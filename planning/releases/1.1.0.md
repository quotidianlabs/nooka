# nooka 1.1.0

Your data, now portable. This release adds full backup and restore, so you can
move your to-do list to a new device or keep an off-device copy — while staying
fully local: no account, no cloud, no sync.

## Highlights

- **Export your data.** Settings → Export data writes your whole list — every
  category and task, archived ones included — to a JSON file you can share or
  save anywhere.
- **Import a backup.** Settings → Import data loads a backup back. After a
  confirmation, it replaces the current list with the file's contents, so a
  restore reproduces your data exactly.

## Notes

- Import **replaces** all current data (behind a confirm dialog) rather than
  merging — it's a restore, not a sync.

Sideload the attached `nooka-1.1.0.apk` on Android.
